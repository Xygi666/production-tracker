# Учёт продукции

Небольшое офлайн‑приложение (PWA) для личного учёта произведённой продукции, брака и расчёта зарплаты. Работает прямо в браузере, данные хранятся локально в `localStorage`.

Демо: https://xygi666.github.io/production-tracker/

---

## Возможности

### Учёт смены

- Добавление записей по продуктам с количеством (поддерживаются отрицательные значения для брака).
- Отдельная цена для брака по каждому продукту.
- Быстрые пресеты количества (1 / 5 / 10 / 25 / 50 и любые свои в настройках).
- Автоподсчёт суммы по выбранному продукту и количеству.
- Отметка «Занесено в 1С» для каждой записи, визуальное выделение уже занесённых:
  - фиолетовый фон и зелёная «галочка».
- Фильтр **«Только незанесённые»** — показывает только те записи, которые ещё не внесены в 1С.
- Группировка записей по дням с заголовком вида `9 марта 2026 г.`.

### Статистика

Вкладка **«Статистика»** показывает срез за текущий месяц.

**Блок «Финансы»:**

- Доход (выручка) за месяц.
- Оклад.
- Налог (по заданной ставке).
- Аванс.
- Итоговая сумма «на руки».

Эти же значения кратко дублируются в шапке приложения.

**Блок «Продукция»:**

Для каждого продукта за текущий месяц:

- количество сделанной продукции;
- количество брака и процент брака;
- суммарная выручка по продукту.

### История по месяцам

Вкладка **«История»**:

- список месяцев (март 2026, февраль 2026 и т.д.) с количеством записей и выручкой;
- данные по месяцу попадают в историю при экспорте CSV;
- при открытии месяца:
  - показываются все записи за месяц, сгруппированные по дням;
  - внизу отдельно выводится сводка **«Продукция за месяц»** в том же формате, что и в статистике.

### Продукты

- Список продуктов с ценой и опциональной ценой брака.
- Добавление, редактирование, архивирование и удаление продуктов.
- Нельзя удалить продукт, если по нему есть записи — можно только отправить в архив.
- Флаг «Избранное»:
  - избранные продукты выводятся выше в подсказках поиска;
  - можно быстро включать/выключать из списка.

### Настройки

В модальном окне **«Настройки»** несколько вкладок:

- **Зарплата**
  - оклад;
  - ставка налога (%);
  - аванс.
- **Продукты**
  - управление списком продуктов;
  - импорт продуктов из CSV (название + цена).
- **Общие**
  - выбор темы (классическая, тёмная, светлая);
  - пресеты количества (набор чисел через запятую).
- **Бэкап**
  - кнопка «Email» — формирует письмо с JSON‑бэкапом данных в теле письма.

### Экспорт и бэкап

- **Экспорт CSV** — выгружает все записи текущего месяца в CSV с колонками:
  - дата;
  - продукт;
  - количество;
  - цена;
  - сумма;
  - флаг «В 1С».
- Экспорт CSV одновременно создаёт «снимок месяца» и сохраняет его в историю.
- JSON‑бэкап можно отправить себе на почту через раздел «Бэкап».

### PWA / офлайн

- Приложение можно установить как PWA (иконка на рабочем столе / в меню приложений).
- Работает без интернета, все данные живут в `localStorage`.
- Обновления подтягиваются через `service worker` при перезагрузке.

## Технологии

- Чистые HTML, CSS, JavaScript.
- Без сторонних библиотек и сборщиков.
- Хранение данных: `localStorage`.

---

## Запуск

Локально:

1. Склонировать репозиторий.
2. Открыть `index.html` в браузере (или поднять простой статический сервер).
3. Для обновления PWA в браузере может понадобиться «жёсткая» перезагрузка и/или сброс service worker.

Приложение также развёрнуто на GitHub Pages и готово к использованию «из коробки».

---

## Планы / идеи

- Счётчик незанесённых в 1С записей и суммы в шапке.
- Сортировка «Продукции» по выручке, браку, количеству.
- Отдельный режим просмотра только брака.
- Кнопка «зафиксировать месяц» в истории.
