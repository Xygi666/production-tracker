# 📊 Учёт продукции

Лёгкое офлайн‑приложение (PWA) для личного учёта произведённой продукции, брака и расчёта зарплаты.  
Работает прямо в браузере, данные хранятся локально в `localStorage`.

Демо: [**https://xygi666.github.io/production-tracker/**](https://xygi666.github.io/production-tracker/)

---

## ✨ Основные возможности

### 📝 Учёт смены

- Быстрый ввод записей по продуктам.
- Поддержка отрицательного количества для брака.
- Отдельная цена брака для каждого продукта.
- Пресеты количества (1 / 5 / 10 / 25 / 50 или свои значения).
- Мгновенный пересчёт суммы для выбранного продукта.
- Отметка «Занесено в 1С»:
  - визуальное выделение таких записей (фиолетовый фон, зелёная галочка);
  - переключатель **«Только незанесённые»** для удобной работы с 1С.
- Записи автоматически группируются по дням.

### 📈 Статистика

Вкладка **«Статистика»** показывает срез за текущий месяц и разделена на два блока.

**Финансы**

- Доход (выручка) за месяц.
- Оклад.
- Налог (по заданной ставке).
- Аванс.
- Итоговая сумма **«на руки»**.

Краткая выручка и итог «на руки» дополнительно видны в шапке приложения.

**Продукция**

Для каждого продукта за текущий месяц:

- сколько сделано (шт.);
- сколько брака (шт. и %);
- суммарная выручка по продукту.

### 📋 История по месяцам

Вкладка **«История»**:

- список месяцев с количеством записей и выручкой;
- данные попадают в историю при экспорте CSV;
- при открытии месяца:
  - показываются все записи, сгруппированные по дням;
  - внизу — сводка **«Продукция за месяц»** (количество, брак, выручка по каждому продукту).

### 🧩 Продукты

- Список продуктов с ценой и опциональной ценой брака.
- Добавление, редактирование, архивирование, удаление.
- Нельзя удалить продукт, если по нему есть записи — только архивировать.
- Избранные продукты:
  - отмечаются звездочкой;
  - выводятся выше в подсказках поиска.

### ⚙️ Настройки

Модальное окно **«Настройки»** с вкладками:

- **Зарплата** — оклад, ставка налога (%), аванс.
- **Продукты** — управление списком, импорт из CSV (название + цена).
- **Общие** — тема (классическая / тёмная / светлая), пресеты количества.
- **Бэкап** — кнопка «Email» формирует письмо с JSON‑бэкапом данных.

### 📤 Экспорт и бэкап

- **Экспорт CSV**:
  - выгружает все записи текущего месяца;
  - поля: дата, продукт, количество, цена, сумма, «В 1С».
- При экспорте CSV автоматически создаётся **снимок месяца** и сохраняется в историю.
- JSON‑бэкап можно отправить себе на почту через раздел «Бэкап».

## 📲 Установка и запуск

### В браузере как сайт

1. Откройте в любом современном браузере:  
   `https://xygi666.github.io/production-tracker/`.
2. Сразу начинайте пользоваться — все данные будут сохраняться локально в браузере на этом устройстве.

### Установка через браузер как PWA

Приложение можно установить на компьютер или телефон и запускать как отдельную программу.

1. Откройте демо‑страницу в поддерживаемом браузере (Chrome, Edge и т.п.).
2. Нажмите «Установить приложение»:
   - на десктопе — иконка `+` в адресной строке или пункт меню **Install app / Установить приложение**;
   - на Android — в меню браузера **Добавить на главный экран** или **Установить приложение**.
3. Подтвердите установку.

После этого приложение:

- появляется на рабочем столе / в списке программ;
- работает офлайн, используя `localStorage` конкретного устройства.
