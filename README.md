# Учёт продукции

Небольшое офлайн‑приложение (PWA) для личного учёта произведённой продукции, брака и расчёта зарплаты. Работает прямо в браузере, данные хранятся локально в `localStorage`.

Демо: https://xygi666.github.io/production-tracker/

## Установка как приложения (PWA)

Приложение можно установить на компьютер или телефон и запускать как отдельную программу.

1. Откройте страницу демо в браузере (Chrome, Edge, мобильный Chrome/Edge/Firefox с поддержкой PWA).
2. В адресной строке появится иконка установки приложения:
   - на десктопе — значок `+` или «Установить приложение»;
   - на Android — пункт меню «Добавить на главный экран» / «Установить приложение».
3. Нажмите «Установить».
4. После установки приложение появится:
   - на рабочем столе и в списке программ (desktop);
   - на домашнем экране (мобильный).

Приложение работает офлайн, все данные сохраняются в браузере на этом устройстве.
